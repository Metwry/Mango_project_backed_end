# Scripts Directory / 脚本目录

## 中文

- 定位：存放环境初始化、依赖安装、Web 和 Celery 启停脚本。
- 当前内容：包含 PowerShell、Shell 和 Batch 版本，覆盖 Conda 初始化、依赖安装、服务启动和停止。
- 使用建议：开发环境优先复用这里的脚本，避免手工命令与部署流程分叉。

## English

- Role: contains environment bootstrap, dependency installation, and web/Celery start-stop scripts.
- Current content: PowerShell, shell, and batch variants for Conda bootstrap, dependency installation, and service lifecycle control.
- Recommendation: reuse these scripts in development to stay aligned with the deployment flow.
